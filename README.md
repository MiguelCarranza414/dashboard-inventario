# Dashboard Inventario & Conteo

Este proyecto es un dashboard web simple para visualizar niveles de inventario y el progreso del conteo cíclico. Contiene:

- `index.html`: la página principal del dashboard.
- `styles.css`: los estilos del dashboard.
- `app.js`: la lógica para cargar datos y filtrar la tabla.

Para ver el dashboard, abre `index.html` en tu navegador.
